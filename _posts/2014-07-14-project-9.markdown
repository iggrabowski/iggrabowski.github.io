This project provided a valuable learning experience, allowing me to explore new technologies and work collaboratively to achieve our objectives.<br><br>Technical details:<ul><li>Photogrammetry pipeline for processing smartphone images into a 3D point cloud.</li><li>Integration of OpenCV for image processing and feature extraction.</li><li>Rendering and visualization of the point cloud using OpenGL.</li></ul>Key Technologies: OpenGL, OpenCV.'
